# 📅 Telegram Post Scheduler Bot

Бот для планирования постов в Telegram-канал.

## ⚙️ Установка

1. Установи Python 3.10+

2. Установи зависимости:
```bash
pip install -r requirements.txt
```

3. Открой `bot.py` и замени:
```python
BOT_TOKEN  = "ВАШ_ТОКЕН_СЮДА"     # Токен от @BotFather
CHANNEL_ID = "ВАШ_КАНАЛ_ID_СЮДА"  # Например: @mychannel
TIMEZONE   = "Europe/Moscow"        # Твой часовой пояс
```

4. Добавь бота в канал как **администратора** с правом публикации постов.

5. Запусти бота:
```bash
python bot.py
```

---

## 🤖 Как получить токен

1. Открой @BotFather в Telegram
2. Напиши `/newbot`
3. Следуй инструкциям
4. Скопируй токен в `BOT_TOKEN`

## 📡 Как получить ID канала

- **Публичный канал**: просто `@название_канала`
- **Приватный канал**: используй @userinfobot или @getidsbot

## 📋 Функции бота

| Кнопка           | Действие                              |
|------------------|---------------------------------------|
| 📝 Создать пост  | Запустить мастер создания поста       |
| 📋 Мои посты     | Показать все запланированные посты    |
| ❌ Отменить пост | Удалить запланированный пост          |

## 🔄 Процесс создания поста

```
1. Фото (или пропустить)
      ↓
2. Текст поста
      ↓
3. Дата и время (формат: 25.12.2025 18:30)
      ↓
4. Подтверждение → Пост запланирован!
      ↓
⏰ В назначенное время → Автопубликация в канал
```
